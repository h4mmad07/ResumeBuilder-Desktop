# Resume Builder Application 📄✨

A robust C# desktop application built using Windows Forms that enables users to create, manage, preview, and export professional resumes. The application utilizes a clean, 3-layer architecture and features multiple professionally designed templates.

## 🚀 Features
* **Structured Data Entry:** Seamless forms for personal details, education, experience, and skills.
* **Six Professional Templates:** From Classic Single-Column to Modern Two-Column layouts.
* **JSON Storage:** Save and reload your resume progress anytime without needing a heavy database.
* **Dynamic PDF Export:** Built with the iText PDF Library, featuring automatic text scaling to prevent layout overflow.
* **Image Processing:** High-quality profile picture integration.

## 🛠️ Tech Stack & Architecture
* **Language/Framework:** C# / .NET / Windows Forms
* **Architecture:** 3-Layer Architecture (Presentation Layer, Business Logic Layer, Data Access Layer)
* **Libraries:** iText PDF Library, System.Text.Json, System.Drawing

## 📦 How to Run the App
### For Users (Standalone):
1. Navigate to the `App` folder in this repository.
2. Download `ResumeBuilder.exe`.
3. Double-click to run! (Requires .NET Runtime installed on Windows).

### For Developers (Visual Studio):
1. Clone this repository.
2. Open `ResumeBuilder.sln` in Visual Studio.
3. Build and Run the project (`F5`).
